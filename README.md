# 🌟 Astra Virtual Pet & Assistant
Một con trợ lý ảo siêu cấp xịn xò chạy trên Desktop Windows, được build bằng C# WPF.

## ✨ Tính năng nổi bật
* **Virtual Pet:** Lượn lờ trên màn hình, bám đuôi siêu mượt.
* **Dashboard:** Quản lý Tasks và Notes xịn xò (Kéo thả bóc Note ra Desktop).
* **AI Chat:** Khung chat tương tác tự nhiên.
* **Storm Reminder:** Nhắc nhở giật tung màn hình chống lười biếng.
* **Random Thoughts:** Tự động lảm nhảm trêu chọc chủ nhân

## 🚀 Cách cài đặt
1. Tải file `.zip` ở phần Releases về.
2. Giải nén và chạy file `AstraCosmeris_.exe`.
